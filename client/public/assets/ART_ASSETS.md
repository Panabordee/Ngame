# CipherDeck art assets / ข้อมูล Artwork

## `cipher-card-back.webp`

English: Original project-bound card-back artwork generated for CipherDeck with the built-in OpenAI image generation tool, then resized to 512×768 and converted to WebP for browser delivery. It contains no third-party logo, text, suit symbol, or brand mark.

ไทย: ภาพหลังไพ่ต้นฉบับที่สร้างสำหรับ CipherDeck ด้วยเครื่องมือสร้างภาพของ OpenAI จากนั้นย่อเป็น 512×768 และแปลงเป็น WebP สำหรับใช้งานบน browser ภาพไม่มี logo, ข้อความ, สัญลักษณ์ดอกไพ่ หรือเครื่องหมายการค้าของบุคคลอื่น

Prompt summary / สรุป prompt: A symmetrical premium card back using an abstract cipher seal, geometric linework, crystalline facets, concentric code rings, deep emerald, charcoal, copper, and antique gold. Flat front view, no text, logo, watermark, people, or perspective.

The CSS card mask, rank/color face layout, Joker face, selection state, and reveal treatment are code-native components in `client/src/CardView.tsx` and `client/src/styles.css`.

กรอบไพ่ การแสดงหน้า/สี หน้า Joker สถานะเลือก และสถานะเปิดไพ่ ถูกสร้างด้วย component/CSS ใน `client/src/CardView.tsx` และ `client/src/styles.css`
